# Markdown Full-Featured Practice Guide / Markdown 全功能练习与语法向导

欢迎来到 Markdown 练习文件！本文件旨在全面涵盖 Markdown 的绝大部分核心与高级语法知识点。通过阅读、编辑（查看源码）以及对照练习，你将能够熟练掌握 Markdown 的高效排版技巧。

---

## 1. 标题文本 (Headings)

Markdown 使用不同数量的 `#` 号来表示 1 到 6 级的标题。

# 一级标题 (H1)
## 二级标题 (H2)
### 三级标题 (H3)
#### 四级标题 (H4)
##### 五级标题 (H5)
###### 六级标题 (H6)

> **练习提示**：在多数编辑器中，建议在 `#` 与标题文本之间保留一个空格。

## 2. 文本样式与格式化 (Text Formatting)

通过特定的符号，你可以为文本添加加粗、斜体、删除线等样式。

- **粗体 (Bold)**：使用双星号 `**加粗文本**` 或双下划线 `__加粗文本__` 效果：**这是加粗的文字**。
- *斜体 (Italic)*：使用单星号 `*斜体文本*` 或单下划线 `_斜体文本_` 效果：*这是斜体的文字*。
- ***粗斜体 (Bold & Italic)***：使用三星号 `***粗斜体***` 效果：***这是又粗又斜的文字***。
- ~~删除线 (Strikethrough)~~：使用双波浪线 `~~删除线~~` 效果：~~这是被划掉的错误内容~~。
- 高亮 (Highlight)：部分编辑器支持使用双等号 `==高亮文本==`（GFM 扩展语法）。
- 下划线 (Underline)：Markdown 原生不支持，但可以通过 HTML 标签实现：<u>带有下划线的文本</u>。

## 3. 列表 (Lists)

### 3.1 无序列表
使用减号 `-`、加号 `+` 或星号 `*` 后跟空格：
- 选项一
- 选项二
  - 子选项二 A (前面缩进两个或四个空格)
  - 子选项二 B
- 选项三

### 3.2 有序列表
使用数字后跟一个英文句点 and 空格：
1. 第一步：打开编辑器
2. 第二步：编写 Markdown 代码
   1. 嵌套步骤 A
   2. 嵌套步骤 B
3. 第三步：导出为 PDF 或 HTML

### 3.3 任务列表 (Task Lists / Checkboxes)
常用于项目管理或 To-Do List：
- [x] 已完成的任务 1
- [x] 已完成的任务 2
- [ ] 未完成的任务 3
- [ ] 正在规划的任务 4

## 4. 区块引用 (Blockquotes)

使用大于号 `>` 来表示引用。区块引用可以嵌套。

> 这是一个基础的区块引用。
> 它可以跨越多行，只要每行开头都有大于号。
> 
> > 这是嵌套的二级引用，用于更深层次的补充说明。
> > —— 某位不愿透露姓名的技术专家

---

## 5. 代码表示 (Code)

### 5.1 行内代码 (Inline Code)
使用单反引号 \` 包裹，用于在正文中提及变量、函数名或短命令。
例如：请使用 `pip install markdown` 命令来安装相关库，然后运行 `main.py` 文件。

### 5.2 代码块 (Code Blocks)
使用三个反引号 \`\`\` 包裹，并在开头指定编程语言，以实现语法高亮。

例如 Python 代码块：
```python
def greet(name):
    print(f"Hello, {name}!")
```

## 6. 链接与图片 (Links and Images)

### 6.1 链接 (Links)
使用 `[链接文字](URL "可选的标题")` 语法：
- [百度一下](https://www.baidu.com)
- [GitHub 官网](https://github.com "GitHub")

### 6.2 图片 (Images)
使用 `![替代文本](图片地址 "可选的标题")` 语法：
- 示例图片：![Markdown Logo](https://markdown-here.com/img/icon256.png "Markdown Logo")

## 7. 表格 (Tables)

使用竖线 `|` 分隔列，使用短横线 `-` 分隔表头与内容。可以通过在短横线两侧添加冒号 `:` 来控制对齐方式：
- `:---` 左对齐
- `---:` 右对齐
- `:---:` 居中对齐

| 项目名称 | 状态 | 优先级 |
| :--- | :---: | ---: |
| 仓库重构 | 进行中 | 高 |
| CI/CD 配置 | 未开始 | 中 |
| 编写测试 | 已完成 | 高 |

---

## 8. 高级与扩展语法

### 8.1 数学公式 (LaTeX)
许多 Markdown 解析器（如 GitHub, Jupyter Notebook）支持 LaTeX 公式：
- 行内公式：$E = mc^2$
- 块级公式：
  $$a^2 + b^2 = c^2$$

### 8.3 HTML 标签
Markdown 支持直接书写 HTML。例如，使用 `<details>` 标签折叠内容：
<details>
<summary>点击展开/折叠内容</summary>

这里是折叠的详细信息。
</details>
